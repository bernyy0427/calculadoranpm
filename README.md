## Instalacion

# Uso
Seguir estas instrucciones:

```
// Importar el modulo
const mod = require('.')

// Suma
console.log(mod.suma(2, 8)) => 10

//Resta
console.log(mod.resta(80, 50)) => 30

// Multiplicacion
console.log(mod.multiplicacion(3, 7)) => 21

// Division
console.log(mod.division(15, 3)) => 5
```
